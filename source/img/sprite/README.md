# Папка для оптимизированных векторных иконок

```shell
└── source/
    └── img/
        └── sprite/
            ├── tg.svg
            ├── vk.svg
            └── yt.svg
```

Находящиеся здесь иконки станут одним спрайтом:

```shell
└── build/
    └── img/
        └── sprite.svg         # спрайт
```
